# Budget-app

## Project setup
```
cd frontend

npm install

cd ../backend

npm install
```

### Compiles and hot-reloads for development
```
budget-app\frontend

npm run serve

budget-app\backend

npm run start

```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```
